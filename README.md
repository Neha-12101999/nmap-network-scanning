# nmap-network-scanning
Network scanning project using Nmap

# Objective
To scan a local network and identify active hosts, open ports, and services.

##  Tools Used
- Nmap
- Command Prompt

##  Commands Used
nmap -sn 192.168.1.1
nmap -sS 192.168.1.1
nmap -sV 192.168.1.1
nmap -O 192.168.1.1

##  Results
- Found active devices in network
- Identified open ports and version
- Detected OS
